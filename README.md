**Dottify** is a Django-based web application simulating a music streaming service.  
It allows users to browse albums, create playlists, submit comments and ratings, and provides employees with administrative tools.

---

### Features

- Full-stack Django application with RESTful API and HTML views  
- User authentication and authorization using Django's built-in features  
- Artist and album management for employees  
- Playlist creation, commenting, and rating for customers  
- Responsive front-end using Bootstrap  
- Secure access control, HTTP status codes, and permissions  
- Unit and integration tests covering models, views, routes, authentication, and authorization
